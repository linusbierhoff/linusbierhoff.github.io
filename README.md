# Linus Bierhoff - Personal Website

This is the source code for my personal website, built with [Astro](https://astro.build/) and Vanilla CSS.

## Features
- Modern Dark Mode with Glassmorphism
- Smooth Scroll Reveal Animations
- Responsive Design
- Projects & Papers showcase

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment
The site is configured for GitHub Pages.
